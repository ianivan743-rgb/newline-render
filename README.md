# newline-render

处理数据时对于有换行符（\r\n\r\n）的内容看起来很费劲，为了更好的查看换行内容和数据，编写了一个网页的换行渲染脚本，对数据进性换行渲染，方便查看。

2026-03-25 新建项目，编写换行渲染网页脚本。

2026-03-26 更新项目：支持json格式解析。初版脚本改名为huanhang.txt。

2026-03-31 新增UI显示支持选中高亮定向展示：换行渲染功能支持左右两侧数据选中高亮展示定位。JSON解析功能支持字段 key 级别联动显示：在原始数据区需完整选择 key ，对应在解析区完整显示；在解析区选择数据可以定向到原始区的 key 所在位置。
           新增历史数据储存，支持数据储存15天，最大储存数量为200条。注意⚠️：历史数据储存保存在浏览器 localStorage 储存历史，15 天自动清理；限制：不同浏览器、不同设备不共享；清浏览器数据会丢失。
